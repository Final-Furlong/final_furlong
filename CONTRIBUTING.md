# Contributing

We welcome contributions from *everyone*.

If you'd like to help make Final Furlong better, here are some ways you can contribute:

- by [reporting bugs you encounter](https://github.com/Final-Furlong/final_furlong/issues/new?template=bug_report.md)
- by [suggesting new features](https://github.com/Final-Furlong/final_furlong/issues/new?template=feature_request.md)
- by taking part in [feature and issue discussions](https://github.com/Final-Furlong/final_furlong/issues)
- by adding a failing test for reproducible [reported bugs](https://github.com/Final-Furlong/final_furlong/issues)
- by reviewing [pull requests](https://github.com/Final-Furlong/final_furlong/pulls) and suggesting improvements
- by writing code (no patch is too small! fix typos or bad whitespace)

If you need help getting started, check out the [README](README.md) file for steps that will get you up and
running.
